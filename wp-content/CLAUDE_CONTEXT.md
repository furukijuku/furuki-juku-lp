## プロジェクト概要

学習塾「Furuki塾江東住吉教室」の公式LPをWordPress固定ページテンプレート（PHP）で構築するプロジェクト。

- **技術スタック**: WordPress + カスタムPHPテンプレート + HTML/CSS/JS（フレームワークなし）
- **インフラ**: AWS Lightsail（現行サーバーから移行予定）
- **管理**: GitHub によるバージョン管理
- **開発フロー**: Claude.aiで設計・コード生成 → Cursorで実装・調整

## Claude.aiとの作業継続について

コードの大きな設計変更・新規LP制作はClaude.aiで行い、  
細かい実装・調整・デバッグはCursorで行う運用を推奨します。

新しい作業をClaude.aiで始める際は、このファイルの内容を冒頭に貼り付けると  
コンテキストを引き継いだ状態でスムーズに再開できます。
