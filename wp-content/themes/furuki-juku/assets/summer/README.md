# 夏期講習ページ用アセット

このフォルダに画像を置くと `summer-course-page.php` から自動で読み込まれます。  
ファイル名は **下記のとおり完全一致** させてください。

## 配置場所

```
wp-content/themes/furuki-juku/assets/summer/
```

## 必要なファイル

| ファイル名 | 用途 | 参照元 |
|---|---|---|
| `character-banzai.png` | 小学生コースの塾長キャラ（右側） | ポスター `01_Elementary_Poster` |
| `character-brain-idea.png` | プログラミングコースの塾長キャラ | ポスター `04_Programming_Poster` |

## 推奨仕様

- 形式: PNG（透過あり推奨）
- キャラクター: ポスター原稿と同じピクセルアート素材をそのまま使用
- ファイルが無い場合: ページは表示されますが、該当コースのキャラ画像のみ非表示になります
